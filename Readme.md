## How to run the app

0. Clone the repository:
```bash
git clone https://github.com/ibai-0/SpeSheet.git
```

1. Create a virtual environment:
```bash
python -m venv venv
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
python main.py
```

4. Open the app in your browser:
```
http://127.0.0.1:8050/
```
